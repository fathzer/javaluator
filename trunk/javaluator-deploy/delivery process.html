Consignes de déploiement de Javaluator.
  • Dans le pom de javaluator-deploy, changer les numéros de version dans la config du plugin ant-run.
  • Faire un maven package des projets :
    • javaluator
    • javaluator-examples
    • javaluator-demo.
  • Faire un maven install de javaluator-deploy.
  • Lancer JavaluatorDeployer en mettant la version de la démo dans la propriété système appletVersion.
  • Faire un maven deploy des projets javaluator et javaluator-examples puis transférer les repository de oss vers maven central.
  • Dans le site de sourceforge, donner les attributs "Linux, BSD, etc" au nouveau zip publié.
  • Répondre aux éventuels bugs reports et feature request ouverts.
  • Tagger la version dans svn.

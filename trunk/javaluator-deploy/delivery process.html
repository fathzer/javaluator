Consignes de déploiement de Javaluator.
  • Dans le pom de javaluator-deploy, changer les numéros de version dans la config du plugin ant-run.
  • Faire un maven build des projets :
    • javaluator
    • javaluator-examples
    • javaluator-demo.
  • Faire un maven build de javaluator-deploy.
  • Faire un maven deploy des projets javaluator et javaluator-examples puis transférer les repository de oss vers maven central.
